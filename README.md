## 💡 Descripción del proyecto

**Deep Learning Complete Guide** es un repositorio educativo orientado al aprendizaje progresivo y práctico de los fundamentos de **Deep Learning**.

El objetivo del proyecto es ofrecer una guía estructurada para estudiantes de **Inteligencia Artificial**, **Machine Learning** y **Deep Learning**, integrando explicaciones teóricas, fórmulas clave, ejemplos de cálculo manual, implementaciones en Python/PyTorch y ejercicios tipo examen.

El repositorio está diseñado para servir tanto como material de estudio como recurso de repaso, manteniendo un formato claro, reproducible y adecuado para su consulta desde GitHub.

---

## 🎯 Objetivos

Este repositorio busca ayudar a comprender Deep Learning desde una perspectiva práctica y académica, evitando limitarse a mostrar código aislado.

**Objetivos principales:**

* Presentar los conceptos fundamentales de Deep Learning de forma ordenada.
* Relacionar la teoría matemática con implementaciones reales.
* Mostrar ejemplos paso a paso con cálculos manuales.
* Implementar modelos y técnicas usando Python y PyTorch.
* Incluir ejercicios tipo examen para consolidar los contenidos.
* Facilitar el repaso de temas clave de aprendizaje automático avanzado.

## 📓 Notebooks incluidos

### 01. Introducción al aprendizaje supervisado

Introduce los conceptos fundamentales del aprendizaje supervisado, incluyendo la relación entre datos, modelos, predicciones, funciones de pérdida y métricas de evaluación.

### 02. Optimización y regularización

Presenta los mecanismos que permiten entrenar modelos de aprendizaje automático y Deep Learning, con especial atención al descenso por gradiente, la regularización y el control del sobreajuste.

### 03. Combinación de estimadores

Explica cómo combinar varios modelos para mejorar la estabilidad y el rendimiento predictivo, introduciendo ideas relacionadas con ensembles y agregación de estimadores.

### 04. Redes neuronales

Desarrolla la base conceptual de las redes neuronales artificiales: neuronas, capas, funciones de activación, propagación hacia delante y ajuste de parámetros.

### 05. Redes neuronales profundas con PyTorch

Muestra cómo construir, entrenar y evaluar redes neuronales profundas utilizando PyTorch, conectando la teoría previa con código ejecutable.

### 06. Redes neuronales convolucionales

Introduce las CNNs como arquitectura fundamental para problemas de visión por computador, explicando convoluciones, filtros, mapas de características y pooling.

### 07. Aspectos avanzados, representaciones y GANs

Recoge conceptos más avanzados relacionados con representaciones aprendidas, modelos profundos y fundamentos de redes generativas adversarias.

## 🚀 Instalación y ejecución

### 1. Clonar el repositorio

```bash
git clone https://github.com/Hugo31810/deep-learning-complete-guide.git
cd deep-learning-complete-guide
```

### 2. Crear un entorno virtual

Con `venv`:

```bash
python -m venv .venv
source .venv/bin/activate
```

En Windows:

```bash
python -m venv .venv
.venv\Scripts\activate
```

### 3. Instalar dependencias

Si el proyecto utiliza `pyproject.toml`:

```bash
pip install -e .
```

También pueden instalarse las dependencias principales manualmente:

```bash
pip install torch torchvision numpy pandas matplotlib scikit-learn jupyter
```

### 4. Ejecutar Jupyter Notebook

```bash
jupyter notebook
```

Después, abrir la carpeta `notebooks/` y ejecutar los notebooks en orden.
